What is HTML?

HTML stands for Hypertext Markup Language, and it's used to create websites! All of the elements in a web page are defined in HTML.

Is HTML a programming language?

NO! HTML is a MARKUP LANGUAGE, such as LATeX. It is not a programming language.

What can HTML do?

HTML can display text in many ways, using headers (h1 to h6), paragraphs like these, bolded text, italics, lists, captions and more,
as well as images, links, forms, buttons and much more!
